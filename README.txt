README

This repo is the master build for the 4-Weeks-to-Open- workout app.
MC-Training-Rolodex is the downstream market/public deploy target; see
CLAUDE.md for the repository relationship, deploy pipeline, and full
project conventions (design system, program layout standard, exercise
library rules, etc.) — that file is the source of truth going forward.

The one-time numbered deploy/patch scripts previously listed here
(1-infra-deploy.py through 5-rest-deploy.py, plus assorted *-patch.py
files) have all been run and removed; they are not part of the live
build or CI pipeline (see .github/workflows/).
